# Employee-Attrition-Workforce-Analytics-Dashboard
Employee Retention &amp; HR Analytics Dashboard | Power BI | DAX | Power Query

Business Problem

Employee attrition directly impacts organizational productivity, recruitment costs, and workforce planning. HR departments require actionable insights into employee demographics, compensation, work-life balance, tenure, and career progression to identify the key drivers of employee turnover.
This dashboard enables HR managers to monitor workforce trends, analyze attrition patterns, and make informed decisions to improve employee retention.
